# qblocks-weapp

#### Description
qblocks-weapp是一个微信小程序版的快速积木式 UI 组件库。"qblocks" 意为 "quick blocks"，旨在帮助您快速搭建漂亮且高效的小程序界面。该库提供常用的 UI 布局组件，可通过简单配置和组合实现快速构建界面。
